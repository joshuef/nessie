Component Description
---------------------

A Nessie PageContent Wraps the body content of the page, usually a Nessie Module.

Example Usage
-------------

	<PageContent>
        <Form>
    		<Module title='Module 1'>
    			Module 1 content...
    		</Module>
    		<Module title='Module 2'>
    			Module 2 content...
    		</Module>
        </Form>
	</PageContent>
